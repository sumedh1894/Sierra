Sierra Consulting Group

A client installation that provides rich collaboration integration between Microsoft Outlook.
Requires Microsoft Outlook 2003 or later

Features

Full integration of email, appointments, tasks and address book

  1. Server side file management and integration with My Files on web mail
  2. Connect to other users resources
  3. Configure permissions for folders for other users
  4. Set out of office message
  5. Offline support

Note: The 32-bit version should be installed on systems that have Outlook 32-bit(irrespective to whether the operating system is 64-bit).Install the 64-bit version on systems that have Outlook 64-bit.
